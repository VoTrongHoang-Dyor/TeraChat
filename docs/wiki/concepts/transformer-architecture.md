# Transformer Architecture

## Definition
Deep learning architecture introduced by Vaswani et al. (2017) in "Attention Is All You Need", now dominant in LLM design.

## Key Facts
- Introduced attention mechanism as core component
- Replaced recurrent layers with self-attention
- Foundation for modern LLMs (GPT, BERT, etc.)
