# Large Language Models

## Definition
AI systems trained on massive text datasets to predict the next token in a sequence.

## Key Characteristics
- **Training**: Large-scale text corpora, next-token prediction objective
- **Architecture**: Dominated by Transformer architecture (per [[Transformer Architecture]])
- **Capabilities**: Text generation, summarization, code writing
- **Limitations**: Hallucinations, context window constraints, no true reasoning
